# 🛒 projeto_DB_E-commerce_tecnologia
Este é um sistema de Banco de Dados para um e-commerce de produtos de técnologia desenvolvidos para em MySQL como projeto de estudos.
# 🛒 Banco de Dados - E-commerce de Tecnologia

## 📌 Objetivo

Este projeto tem como objetivo demonstrar conhecimentos em:

- Modelagem de Banco de Dados
- Modelagem Conceitual
- Modelagem Lógica
- Modelagem Física
- SQL
- Relacionamentos entre tabelas
- Consultas SQL
- Integridade Referencial

---

## 🛠 Tecnologias utilizadas

- MySQL
- MySQL Workbench
- SQL

---

## Estrutura do Banco

O banco possui as seguintes entidades principais:

- cliente
- estoque
- estoquelocal
- fornecedor
- pagamento
- produto
- pedido
- vendedor
- pagamento_cliente
- pedido_pagamento
- pedidoproduto
- produtofornecedor
- produtovendedor

# 🛒 Banco de Dados - E-commerce de Tecnologia

## 📌 Objetivo

Este projeto tem como objetivo demonstrar conhecimentos em:

- Modelagem de Banco de Dados
- Modelagem Conceitual
- Modelagem Lógica
- Modelagem Física
- SQL
- Relacionamentos entre tabelas
- Consultas SQL
- Integridade Referencial

- MySQL
- MySQL Workbench
- SQL
